# Project One

Learning Nodejs<br>
A Nodejs project based on [THIS TUTORIAL](https://scotch.io/tutorials/build-a-restful-api-using-node-and-express-4)

## Prerequisites

Make sure you have node.js installed

## Steps
### In cmd

```bash
    npm install
```

### Running project

You can also start a server:

```bash
    node server.js
```
